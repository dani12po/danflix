## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## API Integration

The app integrates with a real streaming API:
- Base URL: `https://zeldvorik.ru/apiv3/api.php`
- Endpoints: Trending, Movies, Series, K-Drama, Anime, Search
- Caching: 5-minute cache for performance

## Build & Deploy

```bash
npm run build
npm start
```
